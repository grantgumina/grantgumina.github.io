Pichur (pronounced Pi-CH-ur) gives Windows users something they've been wanting for a while: a native photobooth application.

Pichur meets the needs of any computer user looking for such an application: it's fun, fast, stable, and very easy to use. Go ahead and look at more cool things Pichur can do below, but it'd probably be quicker to just go ahead and download it by clicking the button to the right. :)

Features

  • Effect Library

    From the "invert" effect to the "pinch" effect, Pichur includes a large effect library that will keep you entertained for hours. However, in the near future Pichur's library will expand, so keep your eyes pealed to see what's next.

  • Upload to Facebook

    Pichur allows users to seemlessly upload their photos to Facebook. All you need to do is authorize Facebook to let Pichur access your photostream.

  • Open source/Free

    Pichur is 100% free and open source. If you want, you can download and build the source code yourself. In fact, go do that! If you come up with anything interesting you should request to have your changes merged into Pichur's main branch.
